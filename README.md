# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**.
This project allows two players to play the classic Tic Tac Toe game in the browser with sound effects, animations, and responsive design.

---

## 🚀 Features

* ❌⭕ **Two-player gameplay (X vs O)**
* 🟩 **Winning boxes highlight**
* 📏 **Automatic winning line animation**
* 🔊 **Sound effects** for click, win, draw, and reset
* 📱 **Responsive design** (works on mobile, tablet, and desktop)
* 🎨 **Smooth UI animations**
* 🔒 **Boxes lock after selection**
* 🤝 **Draw detection**

---



## 🛠️ Technologies Used

* **HTML5** – Structure of the game
* **CSS3** – Styling, animations, and responsive layout
* **JavaScript (Vanilla JS)** – Game logic and interactions

---


## 🎯 How to Play

1. The game starts with **Player X**.
2. Players take turns clicking on empty boxes.
3. The first player to align **three symbols in a row, column, or diagonal wins**.
4. The winning combination is highlighted with a **line animation and color change**.
5. If all boxes are filled and no player wins, the match ends in a **draw**.
6. Click the **Reset Game** button to start a new game.
